# Envoy 源码研究

## win10 下开发 envoy

### 1、配置流程

- win10 启动 wsl 子系统
- windows store 下载安装 ubuntu 最新版
- apt update && apt upgrade 更新系统
- 下载 gcc/g++ 4.8

  - apt install gcc-4.8
  - apt install g++-4.8
  - 再做一下软链接

    ```软链接
    cd /usr/bin
    ln -s gcc-4.8 gcc
    ln -s g++-4.8 g++
    ln -s gcc-4.8 cc
    ln -s g++-4.8 c++
    ```

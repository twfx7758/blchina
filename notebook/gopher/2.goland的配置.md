# vs code 支持 golang

- 如果代码没有提示
  - go get -u github.com/mdempsky/gocode（gocode 是提示插件）
  - settings.json 添加代理配置，支持 go get -v -u 翻墙
  ```
  "http.proxy": "http://59.53.251.3:50889"
  ```
